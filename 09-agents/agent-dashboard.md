# Agent Dashboard

> **Availability:** `In Preview` 👁️
> **Where to find it:** Agents › Agent Dashboard
> **Who uses it:** treasurers, operations leads, and administrators overseeing automation.
> **Permissions required:** to be confirmed — expected to follow `AdminAndSettings.UserManagement` for managing agents (see [Roles & Permissions](../00-getting-started/04-roles-and-permissions.md)).

## Overview
The **Agent Dashboard** will be the home screen of the Agents area: a single place to see **every AI
agent in your organization, what each one is doing right now, and the outcomes it produces**. At a
glance you'll see how many agents are active, how much work they've automated, their success rate,
and the time they've saved — plus a live feed of recent agent actions.

This screen is **in preview** and not yet available. The description below explains what it will
do so you understand the direction; preview screens exist but the capability is not live.

> The numbers used in demos of this screen (for example "42,850 actions this month", "99.7% success
> rate", "186 h saved") are **illustrative examples**, not your data.

## Key concepts
- **Agent** — a specialized AI assistant with a role, a data scope, and a set of actions. See the
  [AI Agents Overview](overview.md).
- **Outcome tracking** — every agent records what it did, when, and the result, so its impact can be
  measured and audited.
- **Human-in-the-loop** — agents prepare and propose; **people approve**. The dashboard reflects
  that anything sensitive still routes through [approvals](agent-builder.md#approvals).

## What you'll be able to do (planned)
When it ships, the Agent Dashboard is expected to let you:

### See all your agents at a glance
- View a **grid of agents**, each showing its name, specialization, current status (online / idle),
  a key metric, and a one-line "what it's doing now".
- See **top-line KPIs** — active agents, actions this period, success rate, and time saved.
- Spot the **Customer Success agent (Alex)** as the orchestrator that connects your systems and
  routes work to the specialized agents.

### Drill into a single agent
- Select an agent to open a **detail panel** showing its **roles & responsibilities**, the
  **modules it's connected to**, its **key metrics**, and a feed of its **recent actions**.
- Start a conversation with the agent or jump to its full logs.

### Follow live activity
- Watch a **live activity feed** of recent actions across all agents — what happened, which agent
  did it, and when.
