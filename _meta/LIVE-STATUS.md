# Definitive Live Status — from platform menu screenshots

**Rule (confirmed by the team):** menu items tagged **Active** are **live** (`Available`); items tagged
**Upgrade** are **In Preview** (not yet GA — available on request). Top-level items with no tag (Home,
Roadmap) are live.

> **Update:** the Reconciliation (Conciliación) module is **In Preview** — the whole module, per the
> team. (An earlier note treated it as a live exception; that no longer applies.) The separate
> **Reconciliation Status report** under *Reporting › Operations* is a different item and remains
> **Active/Available**.

Badges to use: `Available` ✅ (live) / `In Preview` 👁️ (not live). "Upgrade" is a plan label, noted
separately as **Plan: Premium (Upgrade)** only where useful — it does NOT by itself mean live.
